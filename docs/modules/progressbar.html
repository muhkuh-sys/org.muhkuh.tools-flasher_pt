ProgressBar.default_tparam
Default paramter table of the progress bar.
  • mode nil or 'debug' (default: nil)
  • decimals decimal places percent (default: 1)
  • fill (default: "â–ˆ")
  • prefix (default: "Progress:")
  • suffix (default: "Complete:")
  • length of progress bar (default: 50)
  • printEnd final print (default: "\n")
  • ProgressBarFinished (default: false)
  • printLine at the end of each process step (default: "")
  • ProcessAlgo "Simple" or "SingleLine" or "UnequalByteSingleLine" (default: "Simple")
  • totalByteSize for UnequalByteSingleLine algorithm (default: 1)
  • SL_Iteration for SingleLine algorithm (default: 1)
  • SL_Total for SingleLine algorithm (default: 1)
  • repeatProgress repeat the progress bar (default: false)
  • showTotalProgress total progress after prefix symbol (default: false)

ProgressBar:__getSizeProgressBar (ulMax)
Determine the size of the progress bar depending on default size of the term. The default size of the term is set to 80.

Parameters:

  • ulMax maximum value - total number of bytes
ProgressBar:__ProgressBar_callback_progress (ulCnt, ulMax)
Construct and display the progress bar as callback function. Different algorithm can be used: "Simple", "SingleLine" or "UnequalByteSingleLine"

"Simple": a total progress bar for one event.

"SingleLine": multiple events (SLTotal) in one progress bar. The starting iteration is SLIteration.

"UnequalByteSingleLine": multiple events with different known byte size (sum = totalByteSize) in one progress bar.

Parameters:

  • ulCnt counter value - number of processed bytes
  • ulMax maximum value - total number of bytes
