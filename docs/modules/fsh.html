Module fsh

The FlaSHer application.

A helpful tool to write, verify, read and erase data in flash.

Complete the user input by pressing of tab key - only for matched strings. A callback function which is called every time the user presses tab. It returns a list of items that are completions of the current string.

Parameters:

  • tCompletions a completions object of linenoise. All matches are added to this object.
  • strLine given string of the current line

    __getCompletions

Shell:__run_list (tCmd)
The list command activates the list mode and supports the creation and editing of a list of commands. It can process a list of commands directly through the help of the listed auxiliary commands. Particularly noteworthy is the editing of the 'hist' and 'previous' lists. The list 'hist' contains all processed past commands and the list 'previous' is the saved list of the last editing. ATTENTION: to get access to the hints of possible devices, a connection must be established.

Parameters:

  • tCmd table of commands

See also:

  • run
  • __completer
  • __hints
